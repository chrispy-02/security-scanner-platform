About the Project

Vulnerability Detect and Analysis is a web security tool designed to proactively identify and address vulnerabilities in web applications. Using OWASP ZAP automated scanning, it surfaces issues such as SQL injection, cross-site scripting (XSS), and security misconfigurations. With risk-categorized dashboards and exportable reports, it gives teams the insight needed to safeguard their web infrastructure.

Project Origin

Originally developed as a Michigan State University capstone project sponsored by McKesson. This public portfolio version was cleaned, secured, documented, and prepared for deployment by Christopher Nguyen. It replaces the original GitLab CI scanning pipeline with GitHub Actions, removes hard-coded secrets, and adds a safe public demo mode.
